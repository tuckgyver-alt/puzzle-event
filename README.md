## ✔ 기능
- 시간 기준 오름차순 랭킹
- 실시간 TOP5 표시
- 개인정보 수집 동의
- 관리자 페이지
- Netlify Functions 기반 서버리스 저장

## ✔ 배포 방법

1. GitHub 업로드
2. Netlify → New Site from Git
3. Deploy
